Переменные

Переменная - это хранилище каких-либо данных. В python есть много разных типов данных. Ниже представлены 4 основные типы данных.

  1. Integer (int) - число (целое)
    В этом типе данных хронятся обычные числа. Это могут быть любые не дробные числа, например, 2, 3, 0, 19, -5 или 100.
  2. Float (float) - число (дробное)
    В этом типе данных хронятся дробные числа. Это могут быть как и дробные, так и целые числа. 9.5, 3.14, 42.0, -23.87 или -5.0.
  3. String (str) - строка
    В этом типе данных хронится текст. В таком типе данных можно записать абсолютно любой набор символов. "привет", "13", "myName", "Abidne31IJef".
  4. Bool (bool) - логический тип
    В этом типе данных хронится различные логические значения, например, True или False.

Что бы записать значение в какую-либо переменную нужно придумать название этой переменной, знак равно и значение.

Пример кода: 
    num = 3
    num2 = 2.5
    name = "Luka"
    isGameActive = True
