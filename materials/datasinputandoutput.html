Ввод и вывод данных

В python есть специальные функции для ввода вывода данных. 2 самые основные - print и input. Функция print используется для вывода текста, чисел, значений переменный и т.д.

Пример кода: 
    num = 3
    print(12)
    print(num)
    print(True)
    print("Hello, world")
Результат на экране: 
    12
    3
    True
    Hello, world!

Также есть функция input. С ее помощью можно вводить текст, числа и другое. В скобочках можно ввести текст, который будет показываться во время того как пользователь вводит данные.

Пример кода: 
    name = input("Введите свое имя: ")
Результат на экране: 
    Введите свое имя:
