Работа со строками

В python помимо чисел, можно работать со строками. Строки можно конкатенировать, разбивать на части, доставать определенные элементы, шифровать и т.д.

  1. Конкатенация

    Конкатенация - это склеивание двух или более строк. Например, конкатенация строк "микро" и "мир", то есть "микро" + "мир" --> "микромир" или "abcde" + "12345" --> "abcde12345"

  2. Индексация

    Из строки можно доставать определенные элементы по их индексам. Индекс это порядковый номер символа в строке. Нумерация начинается с нуля, например "hello" --> "h0 e1 l2 l3 o4". Логично, что мы можем доставать значение по индексу.

    Пример кода: 
        text = "Hello, world!"
    char1 = text[4]
    char2 = text[7]
    print(char1)
    print(char2)
    print(text[9])
    Результат на экране: 
        o
    w
    r
  3. Срезы

    Помимо того, что можно достать один символ, можно достать и строку. То есть 1, 2, 3, 4 или сколько угодно символов. Для того, что бы получить строку нажно написать название переменной и открыть квадратные скобочки, например string[3: 6]. Будут вырезаны все символа от первого (включительно) значения до второго (не включительно). Также можно указать третие значение - шаг. Шаг - это количество символов, через которое будет браться следующий. По умолчанию стоит 1, но можно изменить на любое другое значение.

    Пример кода: 
        text = "Hello, world!"
    chars = text[1:4]
    print(chars)
    print(text[3:6])
    print(text[:4])
    print(text[2:])
    print(text[::2])
    Результат на экране: 
        ell
    lo,
    Hell
    llo, world!
    Hlo ol!
  4. ASCII таблицы

    У каждого символа в компьютере есть свой порядковый номер. Эти номера записаны в таблицу, которая называется ASCII. Выше представленны первые 127 символов этой таблицы. Номера символа можно получать с помощью функции ord(), а получать символы из числа с помощью функции chr().

    Пример кода: 
        character = chr(97)
    print(character)
    print(chr(49))
    print(chr(65))
    print(chr(72) + chr(105))
    Результат на экране: 
        a
    1
    A
    Hi
