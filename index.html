Руководство по Python

Python - высокоуровневый язык программирования, ориентированный на повышение производительности разработчика и на обеспечение переносимoсти написанных на нем программ.

Python поддерживает императивное, процедурное, структурное программирование, ООП, метапрограммирование и функциональное программирование.

На python легче писать код, чем на других языках за счет того, что его синтаксис минималистичен, легко читаем и удобен. Благодаря этому, во время написания кода редко возникает необходимость обращаться к документации.

У языка python огромное сообщество, что подтверждается наличием большого количества модулей, библиотек, а также фреймворков.
